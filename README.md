# Typst Plain template

![Thumbnail](thumbnail.png)

Minimal Typst template for MyST/Curvenote: full width, no sidebar. Running header with configurable text (default: "Generated by Curvenote") and logo (`logo.svg`). Used by the task-converter package for Word→PDF conversion.

## Files

- **template.typ** – Main template. Imports `plain.typ`; uses jtex placeholders: `[-doc.title-]`, `[-doc.short_title-]`, `[-IMPORTS-]`, `[-CONTENT-]`. Honors `options.show_running_header`.
- **plain.typ** – Layout and `template()` show rule (margins, running header, title, body). Required by `template.typ`; must be in the same directory when building.
- **template.yml** – jtex v1 config: `doc.title`, optional `doc.short_title`, and `options.show_running_header`.
- **logo.svg** – Logo in the running header (replace with your own if desired).

## Usage

Reference the template by path in your document front matter. When the template folder is next to your project (e.g. `typst-plain/` as a sibling), use:

```yaml
---
title: "My document"
exports:
  - format: typst
    template: ../typst-plain
    output: exports/paper.pdf
    show_running_header: true
---
```

Optional front matter:

- **short_title** – Text in the running header (default: "Generated by Curvenote").
- **show_running_header** – Set to `false` to hide the header and logo on each page.

Example with options:

```yaml
---
title: "My document"
exports:
  - format: typst
    template: ../typst-plain
    output: exports/paper.pdf
    show_running_header: true
short_title: "My Short Title"
---
```

## Comparison to LaPreprint

Based on [myst-templates/lapreprint-typst](https://github.com/myst-templates/lapreprint-typst). This template removes: 25% left margin/sidebar, abstract/affiliations/venue/footer blocks, margin content (kind, dates, acknowledgements). It keeps only title, full-width body, and a minimal running header (short_title + logo).
